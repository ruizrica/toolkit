# Rollout and Validation

## Validation Steps

1. Verify live plan rendering with Mermaid and long structured content.
2. Verify live multi-document spec behavior with comments and raw editing.
3. Verify standalone export for plan and spec.
4. Implement completion and reports live viewers.
5. Verify standalone export for completion and reports.

## Rollout Strategy

- Ship plan/spec export support first.
- Ship completion/reports viewers next.
- Finalize unified docs and examples after all four families are live.
